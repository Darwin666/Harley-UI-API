## CarPlateEditor 车牌号编辑器

用于编辑车牌号。



### Attributes

| 参数            | 说明   | 类型   | 可选值 | 默认值 |
| :-------------- | :----- | :----- | :----- | :----- |
| value / v-model | 绑定值 | String | -      | -      |
| placeholder     | 占位符 | String | -      | -      |



### Events

| 事件名称       | 说明                  | 回调参数                      |
| :------------- | :-------------------- | :---------------------------- |
| change         | 选中值发生变化时触发  | 目前的选中值                  |
| visible-change | 下拉框出现/隐藏时触发 | 出现则为 true，隐藏则为 false |



### Methods

| 方法名   | 说明       | 参数 |
| :------- | :--------- | :--- |
| validate | 校验车牌号 | -    |
