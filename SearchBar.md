## SearchBar 搜索栏

有搜索表单的块状区域。

自带el-form，用于包裹搜索表单项（el-form-item）。



### Attributes

| 参数       | 说明             | 类型   | 可选值 | 默认值 |
| :--------- | :--------------- | :----- | :----- | :----- |
| model      | 表单数据对象     | Object | -      | -      |
| labelWidth | 表单域标签的宽度 | String | -      | 120px  |
| formStyle  | 表单样式         | String | -      | -      |



### Events

| 事件名称 | 说明               | 回调参数 |
| :------- | :----------------- | :------- |
| search   | 点击搜索按钮时触发 | -        |
| reset    | 点击重置按钮时触发 | -        |



### Scoped Slot

| name | 说明     |
| :--- | :------- |
| -    | 内容区域 |
