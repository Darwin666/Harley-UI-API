## Block 块

用包含边框的块状区域来划分页面结构，可包含标题栏区域。



### Attributes

| 参数  | 说明         | 类型   | 可选值               | 默认值 |
| :---- | :----------- | :----- | :------------------- | :----- |
| title | 标题         | String | -                    | -      |
| icon  | 标题前的图标 | String | element-ui提供的图标 | -      |



### Scoped Slot

| name | 说明     |
| :--- | :------- |
| -    | 内容区域 |
