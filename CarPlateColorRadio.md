## CarPlateColorRadio 车牌颜色选择器

用于选择车牌颜色。



### Attributes

| 参数            | 说明   | 类型   | 可选值 | 默认值 |
| :-------------- | :----- | :----- | :----- | :----- |
| value / v-model | 绑定值 | String | -      | -      |



### Events

| 事件名称 | 说明                 | 回调参数     |
| :------- | :------------------- | :----------- |
| change   | 选中值发生变化时触发 | 目前的选中值 |
